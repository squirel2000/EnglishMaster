# dictionary-lookup：單字與片語查詢

## MODIFIED Requirements

### Requirement: 例句數量保證
系統 SHALL 將資料來源為各義項提供的例句歸屬至該筆釋義（每義項至多 1 句，附繁體中文翻譯）。顯示中的釋義所帶例句總數不足 2 句時，系統 SHALL 以 Tatoeba API 取得補充例句，並依釋義顯示順序依序指派給尚無例句的釋義，使其呈現方式與義項自帶例句一致；補充例句 SHALL 與義項例句以英文句去重，且每句 SHALL 附繁體中文翻譯。指派完成後系統不得保留任何獨立於釋義之外的例句清單。單句翻譯失敗時該句 SHALL 退回僅顯示英文，不影響其他例句與查詢結果。

#### Scenario: 義項例句充足
- **WHEN** 顯示中的釋義帶有 2 句以上義項例句
- **THEN** 各例句顯示於所屬釋義之下（英文原句＋繁體中文翻譯），不呼叫補充來源

#### Scenario: 義項例句不足時依序指派補充例句
- **WHEN** 顯示中的釋義僅帶 0-1 句義項例句
- **THEN** 系統向 Tatoeba API 查詢補充例句（與義項例句去重），依釋義顯示順序指派給尚無例句的釋義，呈現方式與義項自帶例句相同

#### Scenario: 補充例句數量少於待指派的釋義數
- **WHEN** 取得的補充例句數量少於尚無例句的釋義數量
- **THEN** 系統依序指派現有補充例句，其餘釋義維持無例句，不因此中斷查詢

#### Scenario: 補充來源亦失敗
- **WHEN** 義項例句不足且 Tatoeba API 查詢失敗
- **THEN** 系統顯示既有的義項例句（可能少於 2 句），不因補充失敗而中斷查詢

#### Scenario: 單句翻譯失敗
- **WHEN** 某個例句的翻譯呼叫失敗，其餘例句翻譯成功
- **THEN** 失敗的例句僅顯示英文，其餘例句顯示英中對照
