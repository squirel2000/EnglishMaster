# dictionary-ui-presentation：深色主題與結果卡片呈現

## MODIFIED Requirements

### Requirement: 字典查詢結果卡片呈現
系統 SHALL 將字典查詢結果呈現為單一結果卡片，依序包含：headword、音標（若有）、發音按鈕、編號釋義（每筆含詞性標示，繁體中文翻譯為主、英文原文對照；無中文翻譯時僅顯示英文）、例句（若有，英文原句＋繁體中文翻譯對照；單句無翻譯時僅顯示英文）、同義字詞與反義字詞（若有）、常用片語（若有）。

#### Scenario: 顯示完整字典結果
- **WHEN** 查詢 "give up" 並取得釋義、例句、同義反義與相關片語
- **THEN** 結果卡片依序顯示 headword「give up」、發音按鈕、編號雙語釋義、雙語例句、同義／反義字詞區塊與常用片語區塊

#### Scenario: 無音標或例句時的卡片呈現
- **WHEN** 查詢結果缺少音標或例句
- **THEN** 結果卡片省略對應區塊，其餘內容照常顯示

#### Scenario: 無同義反義或片語資料時的卡片呈現
- **WHEN** 查詢結果的同義字詞、反義字詞或常用片語為空清單
- **THEN** 結果卡片省略對應區塊，不顯示空標題

#### Scenario: 釋義無中文翻譯時的呈現
- **WHEN** 某筆釋義的繁體中文翻譯不可用
- **THEN** 該筆釋義僅顯示英文原文，其餘各筆照常雙語呈現
