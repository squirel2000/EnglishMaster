# lookup-query：統一查詢入口

## ADDED Requirements

### Requirement: 範例查詢建議
系統 SHALL 在尚未執行任何查詢的待查詢狀態下，顯示至少一個可點擊的範例查詢建議；點擊建議 SHALL 立即以該範例文字執行查詢，行為與使用者手動輸入並送出相同。

#### Scenario: 顯示範例建議
- **WHEN** 使用者開啟應用程式且尚未查詢過任何內容
- **THEN** 系統顯示可點擊的範例查詢建議清單

#### Scenario: 點擊範例建議觸發查詢
- **WHEN** 使用者點擊某個範例查詢建議
- **THEN** 系統以該建議文字執行查詢，並顯示對應的字典釋義或整句翻譯結果
