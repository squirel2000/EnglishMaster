# dictionary-ui-presentation：深色主題與結果卡片呈現

## MODIFIED Requirements

### Requirement: 字典查詢結果卡片呈現
系統 SHALL 將字典查詢結果呈現為單一結果卡片，依序包含：headword、音標（若有）、發音按鈕、加入 Anki 按鈕、編號釋義、更多例句（僅有補充例句時）、同義字詞與反義字詞（若有）、常用片語（若有，每筆附繁體中文解釋；無解釋時僅英文）。每筆編號釋義 SHALL 呈現：繁體中文釋義為主、英文原文緊接其後（無中文翻譯時僅顯示英文），該義項若有歸屬例句則於次行顯示例句（英文原句＋繁體中文翻譯；無翻譯時僅英文）。

#### Scenario: 顯示完整字典結果
- **WHEN** 查詢 "give up" 並取得帶義項例句的釋義、同義反義與相關片語
- **THEN** 結果卡片依序顯示 headword、發音與 Anki 按鈕、編號釋義（各筆為中文釋義＋英文原文＋次行義項例句）、同義／反義字詞區塊與附中文解釋的常用片語區塊

#### Scenario: 義項無歸屬例句時的呈現
- **WHEN** 某筆釋義沒有歸屬例句
- **THEN** 該筆僅顯示雙語釋義，不顯示空例句行，其餘各筆照常

#### Scenario: 有補充例句時的呈現
- **WHEN** 查詢結果包含 Tatoeba 補充例句
- **THEN** 釋義列表之後顯示「更多例句」區塊；無補充例句時不顯示該區塊

#### Scenario: 無音標或例句時的卡片呈現
- **WHEN** 查詢結果缺少音標或任何例句
- **THEN** 結果卡片省略對應區塊，其餘內容照常顯示

#### Scenario: 無同義反義或片語資料時的卡片呈現
- **WHEN** 查詢結果的同義字詞、反義字詞或常用片語為空清單
- **THEN** 結果卡片省略對應區塊，不顯示空標題

#### Scenario: 釋義無中文翻譯時的呈現
- **WHEN** 某筆釋義的繁體中文翻譯不可用
- **THEN** 該筆釋義僅顯示英文原文，其餘各筆照常雙語呈現
