# dictionary-lookup：單字與片語查詢

## MODIFIED Requirements

### Requirement: 例句數量保證
系統 SHALL 將資料來源為各義項提供的例句歸屬至該筆釋義（每義項至多 1 句，附繁體中文翻譯）；顯示中的釋義所帶例句總數不足 2 句時，系統 SHALL 以 Tatoeba API 補充為獨立的補充例句（不歸屬任何義項），使義項例句與補充例句合計 2-3 句。補充例句 SHALL 與義項例句以英文句去重，且每句 SHALL 附繁體中文翻譯；單句翻譯失敗時該句 SHALL 退回僅顯示英文，不影響其他例句與查詢結果。

#### Scenario: 義項例句充足
- **WHEN** 顯示中的釋義帶有 2 句以上義項例句
- **THEN** 各例句顯示於所屬釋義之下（英文原句＋繁體中文翻譯），不呼叫補充來源

#### Scenario: 義項例句不足時補充
- **WHEN** 顯示中的釋義僅帶 0-1 句義項例句
- **THEN** 系統向 Tatoeba API 查詢補充例句（與義項例句去重），合計顯示 2-3 句

#### Scenario: 補充來源亦失敗
- **WHEN** 義項例句不足且 Tatoeba API 查詢失敗
- **THEN** 系統顯示既有的釋義與義項例句（可能少於 2 句），不因補充失敗而中斷查詢

#### Scenario: 單句翻譯失敗
- **WHEN** 某個例句的翻譯呼叫失敗，其餘例句翻譯成功
- **THEN** 失敗的例句僅顯示英文，其餘例句顯示英中對照

### Requirement: 常用相關片語
系統 SHALL 以 Datamuse API 查詢以該字詞開頭的常用片語（依頻率排序，至多 6 筆），每筆片語 SHALL 附繁體中文解釋；單筆片語翻譯失敗時該筆 SHALL 退回僅顯示英文。片語查詢失敗、逾時或無結果時 SHALL 回傳空清單，由呈現層省略該區塊，不得影響查詢結果其餘內容。

#### Scenario: 取得常用片語
- **WHEN** 使用者查詢 "give" 且 Datamuse 回傳片語結果
- **THEN** 系統回傳如 give in、give up、give away 等常用片語（至多 6 筆），每筆附繁體中文解釋

#### Scenario: 片語翻譯失敗
- **WHEN** 某筆片語的翻譯呼叫失敗，其餘片語翻譯成功
- **THEN** 失敗的片語僅顯示英文，其餘片語顯示英中對照

#### Scenario: 片語來源失敗
- **WHEN** Datamuse API 逾時或回應錯誤
- **THEN** 系統回傳空清單，字典查詢結果照常呈現
